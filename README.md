## Features.

- 변경 감지 기능을 활용하여 준영속성 엔티티의 데이터 업데이트 과정을 최적화
- Spring Data JPA의 `JpaRepository`를 상속받아 내장 메서드를 통해 개발의 편리성 제고
- JPA의 EntityManager를 활용하여 직접 쿼리를 구성함으로 데이터 처리의 유연함 제고
- 구조화 된 표현으로 셀 관게 정의
- `DataInitializer`을 통한 사용자별 만다라트 분류
